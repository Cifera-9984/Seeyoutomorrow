# SEE YOU TOMORROW♪

> 昔涟的个人主页，分享我的项目与生活

## 预览

访问 [GitHub Pages](https://cifera-9984.github.io/Seeyoutomorrow/) 查看在线预览。

## 功能特性

- **欢迎进场动画** - 带有流光效果的 WELCOME 进场动画，进度条与弹跳点动画
- **实时天气卡片** - 基于 IP 定位自动获取当前位置天气，支持简略/详情双模式、未来天气预报及空气质量显示
- **背景幻灯片** - 随机切换背景图片，PC 与移动端分离适配，带淡入淡出过渡效果
- **樱花飘落特效** - 纯 CSS 实现的樱花粒子动画，移动端自动减少粒子数量以优化性能
- **玻璃态设计** - 全站采用 Glassmorphism 玻璃态风格，磨砂背景与半透明卡片
- **平滑滚动** - 基于 Lenis 的惯性滚动体验
- **网站设置面板** - 支持自定义主题色相、文字颜色、背景轮播开关、樱花特效开关、文字阴影开关、欢迎动画开关
- **响应式设计** - 完美适配桌面端与移动端
- **自定义字体** - 使用预加载的自定义字体，减少加载闪烁

## 项目页面

- **首页** (`index.html`) - 个人介绍、实时天气、项目展示、联系方式
- **Minecraft 服务器** (`server.html`) - Forge 模组服务器详情页
- **翁法罗斯一年历** (`weng-year.html`) - 崩坏：星穹铁道主题日历
- **钓鱼小游戏** (`fishing_game.html`) - 简单的钓鱼互动小游戏

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: Glassmorphism 玻璃态设计、CSS Grid/Flexbox 布局
- **动画**: CSS Animations, FLIP 动画技术
- **滚动**: Lenis Smooth Scroll
- **图标**: Font Awesome 6
- **API**: IP 定位 API、天气 API

## 本地运行

直接在浏览器中打开 `index.html` 文件即可预览网站。

## 部署

1. 将仓库克隆到本地
2. 推送到 GitHub 仓库
3. 在 GitHub 仓库 Settings → Pages 中开启 GitHub Pages
4. 选择分支和目录，保存后即可访问
